# myip（日本語）

ローカルIPアドレスとネットワークインターフェース情報を表示するシンプルで美しいコマンドラインツールです。

[English](README.md) | [中文](README_zh.md) | [日本語](README_ja.md)

## 機能特性
- メインIPアドレス（インターネット接続に使用されるIP）を表示し、緑色で強調表示
- IPアドレスとそれに対応するネットワークインターフェース名を表示
- IPアドレスをCIDR形式で表示（例：192.168.1.100/24）
- IPv4およびIPv6アドレスの表示をサポート
- IPアドレスタイプをフィルタリングするためのコマンドラインオプション
- クロスプラットフォーム互換性

## 依存ライブラリ
- `clap` - コマンドライン引数の解析
- `colored` - カラー出力のサポート
- `getifs` - ネットワークインターフェース情報の取得

## インストール方法

### 前提条件
- Rustプログラミング言語がインストールされていること（バージョン1.56.0以上）

### ソースからビルド
```bash
git clone <リポジトリURL>
cd myip
cargo build --release
```

実行可能ファイルは `target/release/myip` に配置されます。

## 使用方法

### 基本的な使用方法（IPv4アドレスのみを表示）
```bash
myip
```

![myipコマンド出力](static/myip.png)

### IPv6アドレスのみを表示
```bash
myip -6
```

![myip -6コマンド出力](static/myip-6.png)

### すべてのIPアドレスを表示（IPv4とIPv6の両方）
```bash
myip -a
```

![myip -aコマンド出力](static/myip-a.png)

### ヘルプ情報を表示
```bash
myip -h
```

![myip -hコマンド出力](static/myip-h.png)

## コマンドラインオプション

| オプション | 説明 |
|------------|------|
| `-6`, `--ipv6` | IPv6アドレスのみを表示 |
| `-a`, `--all` | すべてのIPアドレスを表示（IPv4とIPv6の両方） |
| `-h`, `--help` | ヘルプ情報を表示 |
| `-V`, `--version` | バージョン情報を表示 |

## 出力例

```
Ethernet: 192.168.1.100/24
Wi-Fi: 192.168.10.5/24  <-- メインIP（緑色で強調表示）
Loopback: 127.0.0.1/8
```
