# 实现IP与接口名称匹配并格式化输出

## 需求分析

用户已经可以获取所有网络接口和本地IP地址，现在需要将IP地址与对应的接口名称匹配，并按照格式"Local IP (接口名称): IP地址/子网掩码"输出。

从当前输出可以看到：
- 每个网络接口都有一个index（索引）
- 每个Local IP都有一个括号里的数字，表示对应的接口索引

## 实现方案

1. **获取网络接口**：使用getifs库获取所有网络接口信息
2. **获取本地IP地址**：使用getifs库获取所有本地IP地址
3. **创建接口索引映射**：将接口索引与接口名称创建映射关系
4. **匹配IP与接口**：遍历每个IP地址，通过索引找到对应的接口名称
5. **格式化输出**：按照"Local IP (接口名称): IP地址/子网掩码"格式输出

## 具体实现步骤

1. 修改src/main.rs：
   - 获取所有网络接口，创建索引到名称的映射
   - 获取所有本地IP地址
   - 遍历IP地址，提取接口索引，找到对应接口名称
   - 格式化输出匹配结果

2. 测试程序功能，确保输出格式正确

## 预期输出示例

```
Local IP (WLAN): 192.168.10.6/24
Local IP (Radmin VPN): fdfd::1a8f:473/64
Local IP (vEthernet (Default Switch)): 172.17.0.1/20
```
