# 实现fastfetch风格的完整IP信息显示

## 需求分析

用户希望myip工具显示完整的IP信息，格式为"Local IP (接口名称): IP地址/子网掩码"，例如"Local IP (WLAN): 192.168.10.6/24"。

## 实现方案

1. **添加可靠的网络接口库**：使用`getifs`库来获取完整的网络接口信息，包括名称、IP地址和子网掩码
2. **获取活跃IP**：通过UDP套接字确定当前活跃的IP地址
3. **匹配接口信息**：将活跃IP与对应的网络接口匹配，获取接口名称和子网掩码
4. **格式化输出**：按照"Local IP (接口名称): IP地址/子网掩码"格式输出

## 具体实现步骤

1. 修改Cargo.toml，添加`getifs`依赖
2. 修改src/main.rs：

   * 导入`getifs`库

   * 获取所有网络接口的详细信息

   * 通过UDP套接字获取当前活跃IP

   * 匹配IP地址与对应的接口信息

   * 格式化输出完整的IP信息
3. 测试程序功能和显示效果

## 预期输出示例

```
Local IP (WLAN): 192.168.10.6/24
```

或

```
Local IP (以太网): 192.168.1.100/24
```

## 技术说明

* 使用`getifs`库可以跨平台获取网络接口信息

* 通过UDP套接字确定活跃IP的方法保持不变

* 子网掩码将转换为CIDR格式（如/24）

* 确保正确处理IPv4和IPv6地址（优先显示IPv4）
