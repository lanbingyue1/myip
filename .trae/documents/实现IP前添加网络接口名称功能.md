# 实现IP前添加网络接口名称功能

## 需求分析
用户需要在当前IP地址前面加上对应的网络接口名称，输出格式为"接口名称: IP地址"。

## 实现方案
1. **添加依赖**：在`Cargo.toml`中添加`network-interfaces`库，用于获取网络接口信息
2. **修改代码逻辑**：
   - 使用`network-interfaces`库获取所有网络接口及其IP地址
   - 通过UDP套接字获取当前活跃IP地址
   - 将获取的IP地址与网络接口匹配，找到对应的接口名称
   - 按照"接口名称: IP地址"格式输出结果
3. **错误处理**：确保各种情况下都能正常运行并输出有意义的错误信息

## 具体实现步骤
1. 修改`Cargo.toml`，添加`network-interfaces`依赖
2. 修改`src/main.rs`：
   - 导入`network-interfaces`库
   - 获取所有网络接口信息
   - 获取当前IP地址
   - 实现IP与接口的匹配逻辑
   - 输出带接口名称的IP地址

## 预期输出示例
```
以太网: 192.168.1.100
```
或
```
Wi-Fi: 192.168.1.100
```
